O seletor combinador filho > é usado entre seletores para buscar somente os elementos que correspondem ao segundo seletor e são filhos diretos do primeiro seletor.

Isso pode ser útil quando você tiver elementos profundamente aninhados e quer controlar o escopo de seu estilo.

Use o seletor > para marcar os elementos da lista não ordenada dentro dos elementos nav e use Flexbox para espaçar igualmente os elementos filhos.
